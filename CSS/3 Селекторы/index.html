Для стилизации элементов в языке css используются селекторы. Селектор - это строка, по которой css находит один или множество элементов. В качестве селекторов мы будем использовать следующие вещи:

  • Имя тэга (в качестве селектора используется просто имя тэга)
  • Аттрибут class (в качестве селектора используется значение аттрибута class после точки)
  • Аттрибут id (в качестве селектора используется значение аттрибута id после решетки)
