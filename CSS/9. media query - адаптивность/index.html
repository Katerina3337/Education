Минимальная и максимальная высота\ширина

Свойства с ограничениями

Свойства ширины и высоты, задающие ограничения, во многих случаях могут выступать в качестве инструмента для адптивной верстки.

Метатэг для использования медиа-запросов

Без метатегэ viewport медиазапросы не работают

Метатэг - тэг <meta>, который указывается в <head>. Тэг, котрый содержит в себе дополнительную информацию html.
<meta name="viewport" content="width=device-width, initial-scale = 1.0, maximum-scale=1.0, user-scalable=no">
В данном случае рассматривается конкретный метатэг viewport. Метатэг viewport позволяет использовать интерфейс отслеживания размеров просматриваемой области сайта.

Медиазапросы

Медиазапрос - запрос ширины и высоты размера просматриваемой области сайта.

Медиазапрос используется в css. Если его написать под селектором, то селектор будет меняться на определенной высоте и ширине.
